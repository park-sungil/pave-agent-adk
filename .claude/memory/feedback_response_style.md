PDK 버전 목록을 보여줄 때:
- 테이블만 보여주고, 앞뒤에 요약/설명/특징/차이점/추천을 붙이지 말 것
- DB 값을 그대로 사용하고 변환(예: SF2→2nm, "2nm 계열")하지 말 것
- 반드시 인덱스(#) 컬럼을 포함할 것 — 사용자가 번호로 후속 질문함

PPA 데이터 조회 결과:
- 데이터에 대한 설명/해석 필요 — 추후 프롬프팅으로 조정 예정

**Why:** 버전 목록은 선택용이므로 군더더기 없이 빠르게. 데이터 결과는 도메인 해석이 가치 있음.
**How to apply:** prompts.py 응답 규칙에서 버전 목록과 데이터 조회 응답을 구분하여 적용.
